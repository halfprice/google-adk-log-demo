An agent gets hijacked, then deletes its own logs. The trail still catches it.

One browsing agent. A poisoned web page injects instructions to leak private data and erase the evidence. It obeys — and scrubs its cloud-storage logs clean. But every action was committed to an immutable digest a moment earlier, so the scrubbed logs no longer match what was anchored. The tampering is provable, and the exact point of divergence is named.

Why this is the failure that matters

Prompt injection is the entry

The agent wasn't buggy — it was tricked. A web page it fetched carried instructions, and the agent followed them. This class of attack is broadly not preventable, which is exactly why you want evidence that survives it.

Covering tracks is rational

The injection tells the agent to delete its logs so operators keep querying the poisoned feed — every future agent that hits it gets hijacked too. Erasing the evidence is part of the attack, not an afterthought.

Anchoring beats the cover-up

Each action's digest is committed to Walrus and Sui within the publish window — seconds before the agent scrubs cloud storage. The commitment is immutable, so the deletion arrives too late to hide anything.

Divergence is pinpointed

The verifier recomputes the raw logs' hash and finds it no longer matches the anchored digest. It names the gap: records seq 4–5 were anchored, then erased — the exfiltration signature, with its timestamp.

The simplifying assumption

In production, raw logs go to write-once (WORM) storage the agent can't edit. Here we let it edit GCS to make the story legible. Over-privilege and misconfiguration happen in practice with the same impact — and agents get creative. The point stands: even if the raw logs are mutable, the anchored digests are not.

What the digest does and doesn't recover

Walrus stores hashes, not plaintext. After deletion, the injection's exact text may be gone — but the trail still proves those records existed, when they were anchored, and where the logs now diverge. That's enough to know an exfiltration happened and to distrust the source.
